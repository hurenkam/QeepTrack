Works:

  • Target platforms
    • Macbook/iOS Simulator
    • Macbook/OSX Desktop
    • PC/Android Simulator
    • PC/Windows Desktop
    • iPhone 6s
  • Clock
    • Current time
    • Trip time
    • Remaining time (to waypoint)
  • Compass
    • Heading
    • Course
    • Bearing (to waypoint)
  • Speedometer
    • Current speed
    • Minimum speed
    • Maximum speed
    • Average speed
  • Altimeter
    • Current altitude
    • Minimum altitude
    • Maximum altitude
    • Ascent
    • Descent
  • Distancemeter
    • Trip distance
    • Total distance
    • Remaining distance (to waypoint)
  • Levels
    • Compass calibration
  • Any gauge can be put in any spot
  • Options now work and are persistent over restart (gauge positions also)
  • MapView
    • Scroll & Zoom
    • Follow me
    • Select waypoint
    • Show distance between map center and waypoint
  • Datums
    • WGS84
